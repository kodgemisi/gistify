About Personal Github Access Tokens

You DO NOT need a token for showing a gist.

Why You Need It?

Gistify uses Github personal access tokens to update your gists. You only need to give gist permission to your token.

Without a token you can
  • Show any gist
  • Create anonymous gists
With a token you can
  • Show any gist
  • Create personal gists
  • Update your gists

Getting a Token

You can generate a new personal Github token with following steps

  1. Go to this page
  2. Select only gist permission
  3. Generate the token
  4. Copy and give it to Gistify plugin

Remember that if you lost the token you can regenerate it anytime from the same page.
